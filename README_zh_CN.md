# 目录插件

启用插件后，会在顶栏右侧生成一个插件图标

左键点击插件图标，在当前文档下插入目录列表

右键点击插件图标，显示更多操作

`CTRL + ALT + I` 插入目录 

`CTRL + ALT + O` 插入当前文档大纲

# 1.8.0 版本 (主要更新)

- **移出模板功能**: 移除了旧的模板功能。现在可以通过再次点击插入目录/大纲来更新配置。
- **UI 更新**: 转移了插入笔记本目录和插入子页面目录和大纲功能，可以在目录设置里选择。设置面板重构为三列横向标签页（目录、大纲、构建器）。
- **图标系统优化**: 
    - 目录与大纲支持可选图标（默认关闭）。
    - 针对大纲：开启图标时支持富文本同步，关闭图标时仅同步纯文本以保证链接稳定性。
- **加入表情选择器**: Alt + 点击 思源内置表情，可以选择表情进行更换。
- **文档构建器优化**: 移出了拉取功能。
- **图标支持优化**: 非自带图标显示为思源自带图标，不会显示乱码。
